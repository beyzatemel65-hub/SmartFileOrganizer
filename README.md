# 📂 Smart File Organizer

Smart File Organizer is a desktop application built with Python and Tkinter that automatically organizes files into categories such as Images, Documents, Music and Videos.

---

## ✨ Features

- 📁 Select any folder
- 🚀 Organize files with one click
- 🖼 Images
- 📄 Documents
- 🎵 Music
- 🎬 Videos
- 📦 Other files
- 📝 Automatic log generation
- 💻 Simple graphical interface

---

## 🛠 Technologies

- Python 3
- Tkinter
- pathlib
- shutil
